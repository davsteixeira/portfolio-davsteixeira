Zelda API

An application designed for fans of The Legend of Zelda franchise, allowing them to catalog and explore their favorite titles. Developed in Java with Spring Boot, it is composed of three microservices: user management, game lookup via an external API, and a gateway for service integration. I developed a backend API for an online course platform using Java 18, Spring Boot, Spring Data JPA, and MySQL. The system includes core features such as course creation and deactivation, student enrollment, and a report of the most accessed courses. It enforces clear business rules, leverages native SQL for performance optimization, and follows RESTful architecture principles. The project also includes role-based access control (student/instructor), database versioning with Flyway, and a clean, organized code structure based on best practices.
